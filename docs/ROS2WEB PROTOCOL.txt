CLIENTE => SERVIDOR
El cliente envia un mensaje cualquiera:
{
    type: "MESSAGE",
    data: ...
}

SERVIDOR => CLIENTE
El servidor envia un mensaje cualquiera:
{
    type: "MESSAGE",
    data: ...
}

El servidor envia un mensaje de un topic a todos los clientes:
{
    type: "TOPIC",
    data: {
        topic: "topic",
        name: "name",
        value: ...
    }
}
