SERVIDOR => CLIENTE
Mensajes con datos a mostrar (IMAGE, PEOPLE, AUDIO...):
{
    type: "DISPLAY_DATA",
    data: {
        type: "IMAGE",
        value: ...
    }
}

Respuestas a prompts con ID: 
{
    type: "RESPONSE",
    data: {
        id: uuid,
        value: ...
    }
}

Inicializacion de comandos y lo que sea: 
{
    type: "INIT",
    data. ...
}


CLIENTE => SERVIDOR
Prompt del cliente:
{
    type: "PROMPT",
    data: {
        id: uuid,
        value: ...
    },
};
